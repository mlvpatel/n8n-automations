## Why this collection

Most "n8n workflow" repos are thousand-file dumps — broken JSON, dead APIs, and 2023-era model names. This is the opposite. Forty workflows, each one:

| | |
|---|---|
| ✅ **Runs** | Import-tested against a live `n8nio/n8n` container — 40/40 pass |
| 🔄 **Current** | Latest models (GPT-4o, Claude 4.x, Gemini 2.5) and current n8n nodes & syntax |
| 🔐 **Clean** | Zero credentials, secrets, or personal data — safe to open and import |
| 📁 **Organized** | Grouped by role, so you find what fits your work in seconds |

## ⭐ Featured

**[Deep Research Agent](workflows/ai-agents-rag/host-your-own-ai-deep-research-agent-with-n8n-apify-and-openai.json)**  
A self-hosted research agent that plans, searches the live web via Apify, and synthesizes a cited report — your own private version of Deep Research.

**[RAG Chatbot over Google Drive](workflows/ai-agents-rag/ai-powered-rag-chatbot-for-your-docs-google-drive-gemini-qdrant.json)**  
Ask questions across every document in a Google Drive folder. Embeddings in Qdrant, answers from Gemini, grounded in your files.

**[LinkedIn Lead Engine](workflows/sales-crm/linkedin-lead-scraping-horizondatawave.json)**  
A 93-node pipeline that scrapes, enriches, and scores LinkedIn leads into a sheet — a full SDR workflow, automated.

**[AI Short-Form Video Factory](workflows/content-social/ai-powered-short-form-video-generator-with-openai-flux-kling-and-eleve.json)**  
Idea → script → Flux images → Kling video → ElevenLabs voiceover. An end-to-end Reels/Shorts studio in one workflow.

**[Inbox-to-Sheet Expense Tracker](workflows/finance/extract-spend-details.json)**  
Watches Gmail for receipts, extracts the amounts with AI, and logs every expense to a spreadsheet automatically.

**[Automated n8n Backups](workflows/devops-it/automated-workflow-backups-to-google-drive.json)**  
Version-controls your entire n8n instance to Google Drive on a schedule — insurance for your automations.

## 🚀 Quick start

```text
1.  Download any .json from the categories above
2.  In n8n:  Workflows  →  ⋯  →  Import from File
3.  Open each node and add your own credentials
    (every slot is labelled  "Add your <service> credential")
4.  Activate — and you're live
```

> **No secrets required from this repo.** Every credential is blanked and labelled, so nothing here can leak — you plug in your own keys inside n8n's credential manager.
